An intelligent, AI-powered solution for automating attendance tracking. This application leverages the multimodal capabilities of the Google Gemini API to provide a seamless and accurate way to manage classroom attendance through facial recognition.

## âš™ï¸ How It Works

The magic of this application lies in its innovative use of the Gemini API's multimodal and function-calling capabilities.

1.  **Setup**: A teacher creates a classroom and populates the student roster, uploading a clear reference photo for each student.
2.  **Capture**: To take attendance, the teacher uploads a single photo of the classroom.
3.  **API Request**: The application sends the classroom photo along with the entire student roster (images and names) in a single, multimodal request to the Gemini API.
4.  **AI Analysis**: Gemini meticulously analyzes the classroom image, comparing each detected face against the student roster photos. It is instructed to only declare a match with very high confidence to ensure accuracy.
5.  **Structured Response**: The API returns a clean, structured JSON object containing:
    *   A list of `present` students with their names and bounding box coordinates.
    *   A list of `absent` students.
    *   A list of `unknown` faces with their bounding box coordinates.
    *   A qualitative `engagementSummary`.
6.  **Report Generation**: The frontend parses this JSON data to render the rich, interactive, and easy-to-understand attendance report on the teacher's dashboard.
